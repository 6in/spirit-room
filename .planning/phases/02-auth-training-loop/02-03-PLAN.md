| Boundary | Description |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-12 | Information Disclosure | progress.log に API トークンやプロンプト内容が混入 | mitigate | `claude -p` の出力は LLM 応答であり、`.credentials.json` の内容は出力されない。ログはテスト用一時フォルダ内のみ。Plan 完了後のクリーンアップで削除。 |
| T-02-13 | Denial of Service | PHASE1 または PHASE2 が無限ループ | mitigate | フラグファイルチェックで break する設計（既存ロジック）。ステップ 2 でハング監視を明示し、3 分以上進まなければ手動 abort し Plan 01 を再検査する手順を明記。 |
| T-02-14 | Tampering | テストワークスペースのファイルがコンテナ外プロセスに改変される | accept | 一時ディレクトリのためクリーンアップで対応。 |
