<!-- 現行 run_claude 関数（start-training.sh 24-30 行目） -->

```bash
run_claude() {
    local prompt="$1"
    case "$ENGINE" in
        claude)   claude --allowedTools "Bash,Read,Write,Edit,LS" -p "$prompt" 2>&1 | tee -a "$LOG_FILE" ;;
        opencode) opencode -p "$prompt" 2>&1 | tee -a "$LOG_FILE" ;;
    esac
}
```

問題:
1. `--dangerously-skip-permissions` が無いので Bash ツール実行のたびにパーミッションプロンプトが出て stdin 待機でハング
2. `LS` は v2.1.104 の有効ツール名ではない（無視されるだけだが誤解の元）
3. `Glob`, `Grep` が欠けている（コード探索で多用するツール）

修正後の正しい形（RESEARCH.md Code Examples より）:

```bash
run_claude() {
    local prompt="$1"
    case "$ENGINE" in
        claude)
            claude \
                --dangerously-skip-permissions \
                --allowedTools "Bash,Read,Write,Edit,Glob,Grep" \
                -p "$prompt" \
                2>&1 | tee -a "$LOG_FILE"
            ;;
        opencode) opencode -p "$prompt" 2>&1 | tee -a "$LOG_FILE" ;;
    esac
}
```
